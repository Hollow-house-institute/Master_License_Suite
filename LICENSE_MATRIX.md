# Hollow House Institute — License Matrix

This document maps license classes to artifact types.
Binding terms are defined in the individual license texts.

---

## Artifact Coverage Matrix

| Artifact Type | Default Status | License Required |
|--------------|---------------|------------------|
| Datasets | Restricted | Explicit dataset license |
| Research Papers | Restricted | Research-use or commercial license |
| Documentation | Restricted | Documentation license |
| Standards | Restricted | Reference-only license |
| Code | Restricted | Explicit code license |
| Tooling | Restricted | Explicit tooling license |
| Model Training | Prohibited by default | Written authorization required |
| Derivative Works | Prohibited by default | Derivative license required |
| Redistribution | Prohibited by default | Redistribution license required |

---

## Default Rule

No Hollow House Institute material is open-use by default.

Absence of a license grant does not imply permission.
